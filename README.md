# wbuReadAntoNext

武汉商学院阅读平台（悦读平台）自动翻页油猴脚本，支持内网与 VPN 访问，解放双手挂机刷阅读时长。

## 功能特性

- **自动翻页** — 定时自动点击"下一页"，默认每 30 秒翻页一次
- **悬浮窗控制面板** — 实时显示挂机时长，支持动态调整翻页频率
- **极简模式** — 可关闭悬浮窗，切换为 Toast 提示的极简模式
- **Web Worker 多线程** — 使用 Web Worker 处理定时任务，浏览器后台标签页也能稳定运行
- **路由安全锁** — 自动检测当前页面，离开阅读页时自动停止翻页
- **双网络支持** — 同时适配校内网直连和 WebVPN 地址

## 安装

### 1. 安装 Tampermonkey

在浏览器扩展商店中安装 [Tampermonkey](https://www.tampermonkey.net/)（支持 Chrome / Edge / Firefox）。

### 2. 安装脚本

- 打开 Tampermonkey 管理面板 → 新建脚本
- 将 `ReadAntoNext.md` 中的完整代码粘贴进去并保存

## 使用方法

1. 打开悦读平台的任意书籍阅读页
2. **悬浮窗模式（默认）**：页面右上角会出现控制面板，点击「开始挂机」即可
3. **快捷操作**：在阅读页空白区域单击鼠标，也可以快速切换开始/停止
4. 可在悬浮窗中随时修改翻页间隔秒数，立即生效

## 配置

在脚本顶部的「核心配置区」中可修改：

| 配置项 | 默认值 | 说明 |
|---|---|---|
| `ENABLE_UI` | `true` | 是否开启悬浮窗（`false` 为极简 Toast 模式） |
| `currentIntervalSeconds` | `30` | 默认翻页间隔（秒） |

## 适配地址

| 网络环境 | 地址 |
|---|---|
| 校内网 | `ydpj.wbu.edu.cn:8008` |
| WebVPN | `ydpj-wbu-edu-cn-8008-p.webvpn.wbu.edu.cn:8118` |

## 技术实现

- 使用 **Web Worker** (Blob URL) 创建独立线程处理定时翻页，规避浏览器对后台标签页的节流限制
- 通过 `window.location.hash` 进行路由检测，确保仅在阅读页生效
- 累计时长使用 `Date.now()` 计算，防止浏览器休眠导致的计时偏差
